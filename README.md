# A.I.-Terminal-Assistant-For-Kali
INSTALL:
sudo apt update
sudo apt install -y python3 python3-pip git build-essential
pip install -r requirements.txt
RUN:
python3 floot.py --monitor
python3 floot.py --github <URL>
python3 floot.py --autofix
FEATURES:
• AI-assisted error detection
• Real-time terminal monitoring
• GitHub repo setup helper
• Offline model operation
• Auto-fix queue
• Logging of all errors and suggestions
